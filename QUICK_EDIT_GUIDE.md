# 📋 خطوات سريعة للتعديل

## الملف الذي تحتاجه فقط:
```
📂 src/shared/data/portfolio-config.json
```

---

## الخطوة 1️⃣: افتح الملف
1. اذهب إلى المجلد: `src → shared → data`
2. افتح ملف: `portfolio-config.json`

---

## الخطوة 2️⃣: اختر ما تريد تعديله

### 🎨 إذا تريد تعديل البيانات العامة:
ابحث عن:
```
"general": {
```
وعدّل:
- `heroName`: اسمك
- `subtitle`: تخصصك
- `aboutText`: نبذة عنك
- `profilePicture`: صورتك
- `stats.projectsCount`: عدد المشاريع
- إلخ...

---

### 🛠️ إذا تريد إضافة مشروع جديد:

ابحث عن `"projects"` والحط الكود الجديد:

```json
,{
  "id": 4,
  "title": "اسم المشروع الجديد",
  "shortDescription": "وصف قصير",
  "longDescription": [
    "سطر 1",
    "سطر 2"
  ],
  "coverImage": "src/assets/images/project.png",
  "detailImages": ["src/assets/images/img.png"],
  "tags": ["Flutter", "Firebase"],
  "githubUrl": "رابط جيت هاب",
  "liveUrl": "رابط المشروع"
}
```

**ملاحظة مهمة:** ضع علامة فاصلة `,` قبل الكائن الجديد

---

### 💪 إذا تريد إضافة مهارة جديدة:

ابحث عن `"skills"` والحط:

```json
,{
  "id": 10,
  "name": "اسم المهارة",
  "icon": "src/assets/images/icons/icon.png",
  "type": "local"
}
```

---

### 📖 إذا تريد إضافة خبرة جديدة:

ابحث عن `"experiences"` والحط:

```json
,{
  "company": "اسم الشركة",
  "logo": "src/assets/images/logos/logo.jpg",
  "location": "المكان",
  "role": "منصبك",
  "period": "من - إلى",
  "tasks": [
    "مهمة 1",
    "مهمة 2",
    "مهمة 3"
  ]
}
```

---

## الخطوة 3️⃣: احفظ الملف
- اضغط: `Ctrl + S`

---

## الخطوة 4️⃣: عدّل في الموقع
1. افتح الموقع في المتصفح
2. اضغط: `F5` (تحديث الصفحة)
3. شوف التغييرات! ✨

---

## ⚠️ أشياء لا تنساها:

✅ الفاصلة `,` بين العناصر
✅ الأقواس `{ }` والقوسين `[ ]`
✅ علامات الاقتباس `" "`
✅ احفظ الملف بعد التعديل

---

## مثال عملي:

```json
{
  "projects": [
    {
      "id": 1,
      "title": "Project 1"
    }
    ,← ملاحظ الفاصلة هنا
    {
      "id": 2,
      "title": "Project 2"
    }
  ]
}
```

---

## إذا حصل خطأ:

1. تحقق من الأقواس والفواصل
2. استخدم أداة التحقق من JSON:
   - https://jsonlint.com
3. نسخ الكود والصقه هناك للتحقق

---
